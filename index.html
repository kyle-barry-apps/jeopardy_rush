Jeopardy!Rush is a trivia speed challenge based on Jeopardy! questions. See how many you can get right in 60 seconds. Don't worry about "what is...", and make sure you spell your answers correctly. Ready?

Jeopardy!Rush


I love trivia. And I enjoy combining my interests. So, I built Jeopardy!Rush to practice building a web application while creating something fun to use.

The Jeopardy question data was borrowed from this kaggle dataset. I cleaned up the data to make it usable for this project. You can find the jupyter notebook on my github.

Enjoy!
